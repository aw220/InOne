# 用法


## FOFA

1. Config页面配置邮箱和Key，点击Save保存。
2. 输入搜索语法，点击Search搜索，可以进行翻页查看，PreSearch是预搜索，由于fofa比较慢，会在已获取数据不多时在后台获取数据。
3. 除Cert、Header外的列，双击某一条可以加入语法到搜索框。
4. 在表格host列右键有浏览器打开选项，可以在Config配置常用浏览器的绝对路径。
5. Table Size是表格大小。
